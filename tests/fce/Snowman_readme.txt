--> Installation:
1) In "X:\NFS4\Data\Cars" einen neuen Ordner namens "snmn" erstellen
2) Die Datei "car.viv" in diesen neu erstellten Ordner kopieren
3) Falls vorhanden, die *.qfs-Datei nach "X:\NFS4\Data\FeArt\VidWall" kopieren.


--> Spielen:
Spiel wie gewohnt starten und den Wagen in der Auswahlliste markieren.

--> Installation:
1) Create a new folder called "snmn" in your "X:\NFSHS\Data\Cars"-directory.
2) Copy the "car.viv" to this new folder.
3) If contained in the zip-file, copy *.qfs to "X:\NFSHS\Data\FeArt\VidWall"!


--> In the game:
Run the game as usual and select the appropriate car.
